Project1.소켓을 활용한 채팅 프로그램

socket 통신의 기본 개념을 이용하여 회원 관리 (회원 가입, 회원정보 수정, 회원 상세보기) 기능이 추가된 채팅 프로그램입니다. 채팅방 생성 및 삭제가 가능하고, TCP 양방향 통신을 이용 하여 채팅을 구현했습니다.

  • 회원관리
    • 회원의 존재 여부를 확인하고, 회원 추가, 로그인, 회원 정보 수정, 회원 삭제 등을 처리합니다.
  • 채팅방
    • incoming(JSONObject jsonObject) 메소드와 message(JSONObject jsonObject) 메소드에서 채팅방과 관련된 동작을 수행합니다. incoming은 채팅방에 참여하는 동작을 처리하고, message는 채팅 메시지를 모든 클라이언트에게 전송합니다.
  • 메세지송수신
    • send(String json) 메소드에서 클라이언트에게 JSON 형식의 메시지를 전송합니다. DataOutputStream를 사용하여 메시지를 클라이언트로 전송합니다.
    • receive() 메소드에서 클라이언트로부터 JSON 형식의 메시지를 수신합니다. DataInputStream을 사용하여 클라이언트의 메시지를 읽어옵니다. 읽어온 메시지에 따라 다양한 처리를 수행합니다.
