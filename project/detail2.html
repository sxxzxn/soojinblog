Project2.개인 홈페이지 만들기

Java를 이용하여 개인 홈페이지에서 회원 관리를 구현하는 프로젝트입니다. 이 프로젝트를 통해 HTML, CSS, JavaScript, JSP 등의 웹 기술을 적용시키고, 세션 관리에 대한 이해를 목표로 하였습니다.

MAIN

  • 메인화면

MEMBER

  • 로그인, 로그아웃
    • 헤더 영역에서 로그인와 로그아웃 상태를 구분 가능
  • 회원가입
    • 유효성 검사 후 최종 값만 서버로 전달하며 member파일에 저장되도록 작업했습니다
    • 정규표현식을 사용하여 원하는 관리자가 조건 값으로 아이디가 생성 가능하며, 정규표현식의 값으로 알맞지 않은 경우, 알림창으로 해당 오류를 안내하여 사용자의 편의성을 증진시킬 수 있도록 구현하였습니다.
  • 회원정보 수정 및 탈퇴
    • 회원정보를 수정하기 위해 마이페이지 진입 전 비밀번호를 입력 받아 실제 로그인한 회원인지 확인을 통해 보안성을 올렸습니다.
    • 접근 후에는 상단 텍스트 버튼들을 눌러 보다 손쉽게 해당 페이지에 접근하여 회원 정보 확인 및 수정이 가능합니다.
  • 영상으로 프로젝트를 확인하세요!
